#  Mini Calculator (C Language)

##  Project Overview
The **Mini Calculator** is a console-based application developed using the **C programming language**.  
It performs basic mathematical operations such as **Addition, Subtraction, Multiplication, and Division** using a **menu-driven approach** and **modular programming with functions**.

This project is beginner-friendly and focuses on building strong fundamentals in C programming.

---

##  Why This Project Was Created
This project was created to:
- Understand **functions and modular programming** in C
- Practice **conditional statements and switch cases**
- Improve **logic building and problem-solving skills**
- Learn how to take **user input and display clean output**
- Gain hands-on experience with **Git and GitHub project workflow**

---

##  Features
- Menu-driven user interface
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Modulus
- Uses separate functions for each operation
- Clean and readable output
- Beginner-friendly and well-structured code

---

##  Technologies Used
- **Language:** C
- **Compiler:** GCC
- **Editor:** VS Code
- **Version Control:** Git & GitHub
- **Platform:** Windows (Command Line)

---

##  How to Run the Project

1. Clone the repository:
   ```bash
   git clone <your-repository-link>
2. Compile the program:
   gcc mini_calculator.c -o calculator
3. Run the program:
 ./calculator
